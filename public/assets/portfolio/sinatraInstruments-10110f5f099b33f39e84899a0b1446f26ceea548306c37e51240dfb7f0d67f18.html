My Instruments View on Github

An elegant way for musicians to organize their instrument collection.

I come from a family of musicians. My father and uncle grew up playing guitar at all hours of the day, much to the chagrin of my grandparents. My mother and sister play the piano. Myself, I can play guitar, saxophone, drums, bass, ukulele, and am teaching myself piano. So, yeah, of course I'd develop at least one app about music.

This was my first foray into an actual web app. I utilized the Sinatra framework to create a basic MVC CRUD app. A user can catalogue their instrument collection, adding new instruments with Name and Description properties. There is also basic authentication built into the app via the BCrypt gem, so a user can securely sign up and log in with ease.

Engineered via Sinatra to build an MVC application
Designed a secure database with BCrypt using ActiveRecord associations
